## [1.0.0] — 2026-03-29

### First public release

**Content**
- Complete Quranic text in the Hafs an Asim recitation sourced from Tanzil.net
- Typeset in KFGQPC Uthmanic Hafs V22 font with Amiri as auxiliary font
- Full Waqf and Ibtida marks as provided in the Tanzil Uthmani text

**Structure**
- 114 Surah files + cover, dedication, muqaddima, ahzab index, duaa khatm, backcover
- Reflowable EPUB 3.3 format with RTL Arabic layout

**Navigation**
- Surah index (فهرس السور) with links to all 114 Surahs
- Hizb index (فهرس الأحزاب) with clickable links to all 60 Ahzab

**Division markers**
- Full Hizb (ح), Rub (ر), Nisf (ن), Thalatharbaa (ث) and Thumun (•) markers inline with text

**Typography**
- Ayah numbers in fixed-width rounded rectangles, midline aligned
- Decorative double-line dividers between Surahs
- Gold color scheme (#B8860B) throughout

**Accessibility**
- EPUB 3.3 semantic markup (epub:type on all sections)
- ARIA labels on all division markers and ayah numbers
- Color contrast compliant (WCAG AA) for small text elements

**Validation**
- Validated with Bookery EPUBCheck — 0 errors, 0 warnings
